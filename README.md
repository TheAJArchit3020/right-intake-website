# right-intake-website

**Web-facing product for Right Intake AI calorie tracking.**

## Problem
Diet tracking tools often sacrifice usability for data depth, reducing adoption.

## Solution
This web product emphasizes low-friction photo-based calorie estimation for better retention.

## Architecture
- Landing/product web experience
- Core user onboarding flows
- Integration with Right Intake backend services

## Scale
Optimized for acquisition + activation across nutrition-focused audiences.

## Monetization
Subscription conversion funnel with Stripe-enabled purchases.

- Social proof and transformation stories
- Localized growth pages
